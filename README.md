# Character Explorer 

A React application for browsing Rick and Morty characters, built as a routing assignment to demonstrate React Router DOM concepts.


## Features

- Browse all Rick and Morty characters in a responsive grid
- Search characters by name
- Click any character to view their full details
- Tabbed detail view with Info and Episodes
- Active link highlighting in sidebar navigation
- Back navigation between pages

## Tech Stack

- React (Vite)
- React Router DOM v6
- Tailwind CSS v4
- Rick and Morty API — https://rickandmortyapi.com
